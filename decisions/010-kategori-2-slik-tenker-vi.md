# ADR 010 — Kategori 2 navn: "Slik tenker vi"

**Dato:** 2026-04-20
**Status:** vedtatt

## Kontekst
Kategori 2 manglet navn — arbeidstittel var "Faglig overbevisning".

## Valg
"Slik tenker vi". Konkret, ærlig, lavt prestisjenivå.
Slug: `slik-tenker-vi`.

## Konsekvenser
- Koden og alle docs bruker `slik-tenker-vi` som slug.
- UI viser "Slik tenker vi" som display name.
- Reflekterer tonen vi ønsker på Avia — ikke bombastisk, ikke posørete.
